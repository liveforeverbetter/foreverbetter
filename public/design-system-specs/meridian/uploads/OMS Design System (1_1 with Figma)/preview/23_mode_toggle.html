Mode Toggle

Two-segment pill that switches between Sandbox and Live environments. Track gradient changes between states to telegraph the mode at a glance: subtle purple in Sandbox, vibrant purple→cyan when Live.

Sandbox active (state=1)

Live active (state=2)

Interactive (click to toggle)

Anatomy

Specs from Figma nodes 2149:3156 (Sandbox active) and 2149:3155 (Live active).

Outer pill190 × 34, radius 24, border 1px rgb(234,228,245), padding 4
Track · Sandboxlinear-gradient(180deg, rgb(234,228,245), rgb(246,243,251)) — quiet purple haze
Track · Livelinear-gradient(180deg, rgb(213,196,242), rgb(207,250,254)) — bold purple → cyan, signals live mode
Segment91 × 26, radius 24, padding 8; Fustat 500/14
Active segmentbg #fff, color rgb(20,22,53), shadow 0 1 2 rgba(0,0,0,0.05)
Inactive segmentbg transparent, color rgb(100,112,143)
UsageUsed in the sidebar header to switch environment. Always visible — never hidden behind a menu, since the active mode determines what the rest of the app shows.
