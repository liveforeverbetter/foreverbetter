Select component

Select components allow users to select a single option from a predefined list while keeping the interface compact and easy to scan. They are useful for forms, filters, and settings where users need to choose from multiple options. This component is built from two existing components: the input component and the context menu component. As a result, it inherits their behaviour and state variants, including default, hover, selected, focus, and disabled states. The input field also supports typed input, allowing users to filter available options as they search.
