Tabs

Tabs are used to organize and switch between related content within the same page. They help reduce clutter by grouping information into sections without requiring navigation to a new view. Use tabs when content is closely related and users may need to switch between sections frequently. Clearly label each tab with concise, descriptive names, and use the selected state to indicate the active view. Avoid overloading tabs with too many options to maintain clarity and usability.
