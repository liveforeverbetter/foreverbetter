Lucide icon, recolored via `currentColor`, 1.75 stroke to match the brand. Needs the Lucide UMD script loaded on the page.

```jsx
<Icon name="heart-pulse" size={20} />
<Icon name="moon" color="var(--metric-sleep)" />
```
Common glyphs: heart-pulse · moon · activity · zap · flame · bed · footprints · trending-up · plus · chevron-right.
