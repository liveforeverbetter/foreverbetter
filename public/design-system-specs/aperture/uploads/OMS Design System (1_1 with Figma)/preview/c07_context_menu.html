Context menu

The context menu provides a list of actions related to a specific item or element. It is typically triggered by an overflow icon (three dots) and surfaces secondary actions without cluttering the main interface. Use context menus for item-level actions such as edit, duplicate, or delete. Keep the list concise and prioritize frequently used actions, using visual distinction (e.g. danger styles) for destructive options.
