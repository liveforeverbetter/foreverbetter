Modal

A modal is an overlay component used to focus user attention on a specific task or piece of information without leaving the current page. It appears above the main interface with a dimmed backdrop and typically includes a clear title, supporting content, and primary and secondary actions. The content area is a flexible, customizable slot that can accommodate forms, messaging, or multi-step flows. On desktop, modals are centered with fully rounded corners, while on mobile they appear as a bottom sheet—anchored to the bottom of the screen with straight bottom corners and rounded top corners—so actions are easier to reach with the user's thumbs.

Anatomy

All variants share the same shell: Header (32 key icon · 28/700 title · 12/500 substep) · Body (24-gap stack, optional progress bar at top) · Actions (16-gap row of buttons). Surface = white, 24px radius, 1px slate-200 border, modal shadow, 24px padding all sides. Mobile flips bottom corners to 0 and anchors to the viewport bottom.
