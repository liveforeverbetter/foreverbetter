Circular, icon-only button for chrome and card actions.

```jsx
<IconButton label="Back" variant="ghost"><i data-lucide="chevron-left" /></IconButton>
<IconButton label="More" variant="outline"><i data-lucide="more-vertical" /></IconButton>
```

Use `active` for selected states (applies teal tint). Sizes `sm(34) · md(42) · lg(52)`.
