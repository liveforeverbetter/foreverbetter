Metric/nav row with a tinted circular icon chip, title/subtitle, and a trailing monospace value or chevron.

```jsx
<ListItem icon={<i data-lucide="heart" />} iconTone="heart"
  title="Resting heart rate" subtitle="7-day average" value="58" valueUnit="bpm" chevron />
```
Match `iconTone` to the health domain. Set `divider` on all but the last row in a grouped list.
