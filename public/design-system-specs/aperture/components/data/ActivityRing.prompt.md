Concentric activity rings for the daily-activity summary (steps / active minutes / calories).

```jsx
<ActivityRing size={120} rings={[
  { value: 93, max: 120, color: "var(--activity-500)" },
  { value: 1, max: 30, color: "var(--sleep-500)" },
  { value: 6, max: 20, color: "var(--nutrition-500)" },
]} />
```
Outermost ring is listed first. Keep to 3 rings.
